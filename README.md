# Simple Calculator

A clean and user-friendly calculator web application built with vanilla JavaScript, HTML, and CSS. Perform basic arithmetic operations with an intuitive interface.

## Features

- **Basic Arithmetic Operations** - Addition, subtraction, multiplication, and division
- **Clear Function (AC)** - Quickly reset the calculator
- **Delete Function (DEL)** - Remove the last entered digit
- **Decimal Support** - Perform calculations with decimal numbers
- **Responsive Display** - Shows both current and previous operands
- **Clean UI** - Simple and intuitive button layout
- **Custom Favicon** - Professional branding

## Technologies Used

- **HTML5** - Structure and layout
- **CSS3** - Styling and responsive design
- **JavaScript (ES6)** - Calculator logic using object-oriented programming

## How to Use

1. **Clone or Download** this repository to your local machine
2. **Open** `index.html` in your web browser
3. **Start Calculating**:
   - Click number buttons (0-9) to enter numbers
   - Click operation buttons (+, -, ×, ÷) to select an operation
   - Click the decimal button (.) for decimal numbers
   - Click **=** to see the result
   - Click **AC** to clear everything
   - Click **DEL** to remove the last digit

## Code Highlights

The calculator uses an object-oriented approach with a `Calculator` class that handles:
- Number input and validation
- Decimal point handling (prevents multiple decimals)
- Arithmetic computations
- Display updates
